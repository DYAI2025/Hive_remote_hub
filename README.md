# Agent-Hub

Eine kleine, autonome Steuerzentrale für GPT-gestützte Agentenprojekte.

## Funktionen
- Projekte registrieren
- Wissen verwalten (Markdown)
- REST-API zur Agentenabfrage

## Start
```bash
pip install flask pyyaml
python app.py
```

## API-Endpunkte
- POST /register
- GET /wiki/<projekt>
- GET /agents
